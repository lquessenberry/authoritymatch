# AuthorityMatch — Public Website

This is the public-facing website for **AuthorityMatch**.

## Live Site

https://lquessenberry.github.io/authoritymatch/

## About

AuthorityMatch connects newly authorized domestic truckers with trusted factoring partners through a clean, self-serve experience.

Built as part of the pilot with the Jonesboro Factoring Network.

## Tech

- Single-file static site
- Tailwind CSS (via CDN)
- Font Awesome icons

## Local Development

Just open `index.html` in any browser.

## Deployment

This site is deployed via GitHub Pages.
